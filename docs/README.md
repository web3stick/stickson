# Stickson Documentation

Stickson is a JSON framework for making static HTML pages. Write your content in JSON, validate it, and build static HTML output.

- [Install Guide](INSTALL.md) — install stickson globally or as a dev dependency
- [Tutorial](TUTORIAL.md) — build your first stickson site from scratch
- [CSS Guide](CSS_GUIDE.md) — styling your stickson pages
